DQUERY

Summary​

  • Given an array of n integers, for each of qqq queries [l,r][l,r][l,r], count the number of distinct values in a[l..r]a[l..r]a[l..r].
  • Constraints:
    • 1≤n≤3⋅1041\leq n\leq 3\cdot 10^41≤n≤3⋅104
    • 1≤ai≤1061\leq a_i \leq 10^61≤ai​≤106
    • 1≤q≤2⋅1051\leq q\leq 2\cdot 10^51≤q≤2⋅105

Link to Problem Statement

Solution​

  • Since the queries are offline, sort the queries by (r,l,idx)(r,l,idx)(r,l,idx) and perform a line sweep from left to right
  • Maintain the 0/1 contribution array, but now a value contributes 1 if it is the rightmost occurrence in the range [0,r][0,r][0,r] instead of leftmost
  • Use a Fenwick tree to maintain contribution of each value up to the current query's right endpoint
  • Then for a query [l,r][l,r][l,r] the number of distinct values in the range is simply bit.sum(r) - bit.sum(l-1)
  • As we advance the right pointer, we perform add(ptr, 1) and add(last[a[ptr]], -1) if last[a[ptr]] != -1

Code​

void solve(){
    int n;
    cin >> n;
    vi a(n + 1, 0);
    rep(i,1,n+1) cin >> a[i];
    int q; 
    cin >> q;
    vector<array<int, 3>> queries;
    rep(i,0,q) {
        int l, r;
        cin >> l >> r;
        queries.push_back({r,l,i});
    }
    sort(all(queries));
    int M = *max_element(all(a));
    vi last(M + 1, -1); // last occurrence of a[i]
    vi bit(n + 1, 0);
    vi ans(q);
    auto add = [&](int i, int v) {
        for (; i <= n; i += i & (-i)) bit[i] += v;
    };
    auto pre = [&](int i) {
        int ret = 0;
        for (; i > 0; i -= i & (-i)) { ret += bit[i]; }
        return ret;
    };
    int ptr = 1;
    for(auto& qr : queries) {
        int r = qr[0], l = qr[1], i = qr[2];
        while (ptr <= r) {
            if (last[a[ptr]] != -1) add(last[a[ptr]], -1);
            add(ptr, 1);
            last[a[ptr]] = ptr;
            ptr++;
        }
        ans[i] = pre(r) - pre(l-1);
    }
    for(int x : ans) cout << x << " ";
}

Alternative Solution​

A persistent segment tree can also be used, but it is slightly overkill and slower than using a Fenwick tree.
