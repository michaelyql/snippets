CF1714D - Color with Occurrences

Summary​

  • Given a string ttt and nnn strings s1,…,sns_1,\dots,s_ns1​,…,sn​, in one move you can use any sis_isi​ to cover an occurrence of sis_isi​ in ttt at any index.
  • Each sis_isi​ can be used zero or more times.
  • Strings used to cover can overlap each other.
  • Find the minimum number of moves needed to cover ttt entirely, and output the strings used and the positions they cover.
  • Constraints:
    • 1≤∣t∣≤1001\leq |t|\leq 1001≤∣t∣≤100
    • 1≤n≤101\leq n\leq 101≤n≤10
    • 1≤∣si∣≤101\leq |s_i|\leq 101≤∣si​∣≤10
    • 1≤q≤1001\leq q\leq 1001≤q≤100

Link to Problem Statement

Solution​

  • Let dp[i]dp[i]dp[i] be the min. strings needed to cover t[0..i]t[0..i]t[0..i]
  • Brute force all suffixes t[j..i]t[j..i]t[j..i] ending at iii
  • If there is a match between t[j..i]t[j..i]t[j..i] with some sis_isi​, then set dp[i]=min⁡(dp[i],1+min⁡k=j−1i−1dp[j])dp[i]=\min(dp[i], 1 + \min_{k=j-1}^{i-1} dp[j])dp[i]=min(dp[i],1+mink=j−1i−1​dp[j])
  • Time Complexity: O(∣t∣⋅∑i=1n∣s∣)O(|t| \cdot \sum_{i=1}^n |s|)O(∣t∣⋅∑i=1n​∣s∣) per query

Code​

string t;
int n;
unordered_set<string> strs;
unordered_map<string, int> str_to_id;
vi sz;
const int inf = 1e9;
int m;
vi dp, par, use;

int f(int i) {
    if (i < 0) return 0;
    if (dp[i] != inf) return dp[i];
    int j = i;
    string cur= "";
    int curmin = inf+1;

    int bestdp_i = -1;
    bool found=false;

    while (j >= 0) {
        cur = t[j] + cur;
        int res = f(j-1);
        if (res < curmin) {
            bestdp_i = j-1;
            curmin = res;
        }
        if (strs.find(cur) != strs.end()) {
            if (curmin + 1 < dp[i]) {
                dp[i] = curmin+1;
                par[i] = bestdp_i;
                use[i] = str_to_id[cur];
                found=true;
            }
        }
        j--;
    }
    if (bestdp_i == -1 && !found) return dp[i] = inf+1;
    return dp[i];
}

void solve(){
    cin >> t;
    cin >> n;
    strs.clear();
    str_to_id.clear();
    sz.resize(n, 0);

    dp.clear();
    par.clear();
    use.clear();

    rep(i,0,n){
        string s;
        cin >> s;
        strs.insert(s);
        str_to_id[s]=i;
        sz[i]=(int)s.size();
    } 

    m = (int) t.size();
    dp.assign(m, inf);
    par.assign(m, -1);
    use.assign(m, -1);

    int minsteps = f(m - 1);
    if (minsteps==inf+1) {
        cout << -1 << '\n';
        return;
    }
    else cout << minsteps << '\n';
    int cur = m - 1;
    vector<pii> ans;
    while (cur > -1) {
        int s = sz[use[cur]];
        int start = cur - s + 1;
        ans.pb({use[cur] + 1, start + 1});
        cur = par[cur];
    }
    for(auto [w, p]: ans) cout << w << " " << p << '\n';
}
