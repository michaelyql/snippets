# CF1713D - Tournament Countdown

## Summary

- Given a single-elimination tournament tree with $2^n$ people, in each query you can query two indices $a$ and $b$, and receive 1 if $a$ won more rounds than $b$, 2 if $b$ won more rounds than $a$, or 0 if tied.
- Find the winner in no more than $\lceil \frac{1}{3}\cdot 2^{n+1} \rceil$ queries.
- Constraints: 
    - $1\leq n\leq 17$

[Link to Problem Statement](https://codeforces.com/problemset/problem/1713/D)

## Solution

- You can eliminate three people using two queries for every group of four people
- In a group of people $a,b,c,d$, query $a$ and $c$
- If $a>c$, then only $a$ and $d$ can be winners
- Vice versa, if $a<c$, then only $b$ and $c$ can be winners
- If $a==c$, then both must have lost against their opponents, so $b$ and $d$ are winners
- Spend one more query to determine the winner out of each group of four

void solve() {
    int n;
    cin >> n;
    vi keep((1<<n));
    iota(keep.begin(), keep.end(), 1); 
    for(int i=0;i<n/2;i++){
        vi nxt;
        for(int j=0; j+3<(int)keep.size(); j+=4) {
            cout << "? " << keep[j] << " " << keep[j+2] << "\n";
            cout.flush();
            int res;
            cin >> res;
            if (res==1){
                cout << "? " << keep[j] << " " << keep[j+3] << "\n";
                cout.flush();
                cin >> res;
                if (res == 1) nxt.pb(keep[j]);
                else if (res == 2) nxt.pb(keep[j+3]);
                else {
                    exit(0);
                }
            }else if (res==2){
                cout << "? " << keep[j+1] << " " << keep[j+2] << "\n";
                cout.flush();
                cin >> res;
                if (res == 1) nxt.pb(keep[j+1]);
                else if (res == 2) nxt.pb(keep[j+2]);
                else {
                    exit(0);
                }
            }else if (res==0){
                cout << "? " << keep[j+1] << " " << keep[j+3] << "\n";
                cout.flush();
                cin >> res;
                if (res == 1) nxt.pb(keep[j+1]);
                else if (res == 2) nxt.pb(keep[j+3]);
                else {
                    exit(0);
                }
            }else {
                exit(0);
            }
        }
        keep = move(nxt);
    }
    if ((int)keep.size()==1){
        cout<<"! " << keep[0] << '\n';
        cout.flush();
        return;
    }
    cout << "? " << keep[0] << " " << keep[1] << '\n';
    cout.flush();
    int res;
    cin >> res;
    if (res == 1) {
        cout << "! " << keep[0] << "\n";
        cout.flush();
    } else if (res == 2) {
        cout << "! " << keep[1] << '\n'; 
        cout.flush();
    } else {
        exit(0);
    }
}
