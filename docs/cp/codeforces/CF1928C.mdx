# CF1928C - Physical Education Lesson

## Summary

- Given the infinite sequence $1,2,\dots,k,k-1,\dots,2,1,2,\dots$ that repeats every $2k-2$ numbers, the position $n$ in the sequence and the number $x$ that it lands on, count the number of valid $k$ that satisfies the given $n$ and $x$.
- Constraints: 
    - $1\leq x<n\leq 10^9$

[Link to Problem Statement](https://codeforces.com/problemset/problem/1928/C)

## Solution

- Boils down to solving for $n \% (2k-2) = x$

### Code

```cpp
void solve() {
    int n, x;
    cin >> n >> x;
    unordered_set<int> s;
    if (x == 1) {
        for(int i=1; (ll)i*i <= n-1; i++) {
            if ((n-1) % i == 0) {
                if (i % 2 == 0) s.insert((i+2)/2);
                int d = (n-1) / i;
                if (d % 2 == 0) s.insert((d+2)/2);
            }
        }
    } else {
        if ((n-x) % (2 * x - 2) == 0) s.insert(x);

        for(int i=1; (ll)i*i <= n-x; i++) {
            if ((n-x) % i == 0) {
                if (i % 2 == 0 && x < (i+2)/2) {
                    s.insert((i+2)/2);
                }
                int d = (n-x)/i;
                if (d % 2 == 0 && x < (d+2)/2) {
                    s.insert((d+2)/2);
                }
            }
        }

        for(int i=1; (ll)i*i <= n+x-2; i++) {
            if ((n+x-2) % i == 0) {
                if (i % 2 == 0 && x < (i+2)/2) {
                    s.insert((i+2)/2);
                }
                int d = (n+x-2) / i;
                if (d % 2 == 0 && x < (d+2)/2) {
                    s.insert((d+2)/2);
                }
            } 
        }
    }
    cout << s.size() << '\n';
}
```
