# CF1714F - Build a Tree and That Is It

## Summary

- Build a tree on $n$ nodes such that the distance between $(1,2)$, $(2,3)$ and $(1,3)$ is $d_{12}$, $d_{23}$ and $d_{31}$ respectively, or report that it does not exist.
- Constraints: 
    - $3\leq n\leq 2\cdot 10^5$
    - $1\leq d_{12},d_{23},d_{31}\leq n-1$

[Link to Problem Statement](https://codeforces.com/problemset/problem/1714/F)

## Solution

- Solve for two cases
- Case 1: 1,2,3 are on a single line
    - Occurs when $d_{12}=d_{23}+d_{31}$ (WLOG)
- Case 2: 1,2,3 do not lie on a line
    - Thus there is some "central" node that lies between 1,2 and 3
    - WLOG, let $d_{12}$ be the largest distance out of the three distances
    - Try all points between 1 and 2 as the 'midpoint', and use the distances to verify that it sums correctly
    - Edge case: check that the number of nodes required for that particular mid point does not exceed $n$

## Code

```cpp
void solve(){
    int n, d12, d23, d31;
    cin >> n >> d12 >> d23 >> d31;
    int a = 1, b = 2, c = 3;
    if (d12 < d23) {
        swap(a, c);
        swap(d12, d23);
    }
    if (d12 < d31) {
        swap(b, c);
        swap(d12, d31);
    }
    // now d12 is maximal
    if (d12 == d23 + d31) {
        cout << "YES\n";
        vi ans;
        ans.pb(a);
        for(int i=1;i<=d31-1;i++) ans.pb(i+3);
        ans.pb(c);
        for(int i=1;i<=d23-1;i++) ans.pb(i+d31+2);
        ans.pb(b);
        for(int i=d12+2; i<=n; i++) ans.pb(i);
        for(int i=0;i<(int)ans.size() - 1; i++) {
            cout << ans[i] << " " << ans[i+1] << '\n';
        }
        return;   
    }
    if (d31 + d23 <= d12) {
        cout << "NO\n";
        return;
    }
    int s = -1;
    for(int i=1; i<=d12-1; i++) {
        int k = d31 - i;
        if (d12 - i + k == d23) {
            s = i;
            break;
        }
    }
    if (s == -1) {
        cout<<"NO\n";
        return;
    }
    if (d12 + d31 - s + 1 > n) {
        cout << "NO\n";
        return;
    }
    vector<pii> edges;
    vi ans {a};
    for(int i=1; i<=d12 - 1; i++) ans.pb(3+i);
    ans.pb(b);
    for(int i=0;i<(int)ans.size()-1; i++){
        edges.pb({ans[i], ans[i+1]});
    }
    ans.clear();
    ans.pb(c);
    // d12+3
    for(int i=1; i<=d31 - s - 1; i++) ans.pb(d12+i+2);
    ans.pb(3+s);
    for(int i=0;i<(int)ans.size()-1; i++){
        edges.pb({ans[i], ans[i+1]});
    }
    ans.clear();
    ans.pb(2);
    for(int i=d12+d31-s+2; i<=n; i++) ans.pb(i);
    for(int i=0;i<(int)ans.size()-1; i++){
        edges.pb({ans[i], ans[i+1]});
    }
    cout << "YES\n";
    for(auto& e:edges){
        cout<<e.first<<" " <<e.second<<'\n';
    }
}
```
